\doxysubsection{Detailed Description}
Data structure for UNIT\+\_\+B components. 

Following EPOS Smart\+Data principles, this structure contains all the data needed for both \doxylink{struct_basic_producer_b}{Basic\+ProducerB} and \doxylink{struct_basic_consumer_b}{Basic\+ConsumerB} functionality. Replaces inheritance-\/based data storage with pure composition.

Uses a different default range (200.\+0f to 300.\+0f) compared to UNIT\+\_\+A to provide clear differentiation between component types.
