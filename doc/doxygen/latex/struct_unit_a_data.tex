\doxysubsection{Detailed Description}
Data structure for UNIT\+\_\+A components. 

Following EPOS Smart\+Data principles, this structure contains all the data needed for both \doxylink{struct_basic_producer_a}{Basic\+ProducerA} and \doxylink{struct_basic_consumer_a}{Basic\+ConsumerA} functionality. Replaces inheritance-\/based data storage with pure composition.
