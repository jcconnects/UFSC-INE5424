\doxysubsection{Detailed Description}
This class initializes the API. 

This class is only responsible for starting the API. It allows you to create a network interface (\doxylink{class_n_i_c}{NIC}) and a protocol, which will be used in the system. The idea is that it is independent of the application in which the developer will use the API.
