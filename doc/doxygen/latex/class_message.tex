\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Channel$>$\newline
class Message$<$ Channel $>$}
Template class for network messages with \doxylink{class_clock}{Clock} integration. 

This \doxylink{class_message}{Message} class is integrated with the \doxylink{class_clock}{Clock} singleton to provide synchronized timestamps across the distributed system. When messages are created, they automatically use the \doxylink{class_clock}{Clock}\textquotesingle{}s get\+Synchronized\+Time() method instead of local system time, ensuring temporal consistency in a PTP-\/synchronized network.

\doxylink{class_clock}{Clock} Integration Features\+:
\begin{DoxyItemize}
\item Automatic synchronized timestamping in constructor
\item Static utility methods for getting synchronized timestamps
\item \doxylink{class_clock}{Clock} synchronization status checking
\end{DoxyItemize}
