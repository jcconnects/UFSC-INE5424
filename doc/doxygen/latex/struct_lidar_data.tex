\doxysubsection{Detailed Description}
Data structure for Lidar component. 

Following EPOS Smart\+Data principles, this structure contains all the data needed for \doxylink{struct_lidar_component}{Lidar\+Component} functionality. Generates realistic 3D point cloud data with configurable ranges and densities for simulation purposes.

Replaces inheritance-\/based data storage with pure composition.
