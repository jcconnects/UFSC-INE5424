\doxysubsection{Detailed Description}
Data structure for ECU component. 

Following EPOS Smart\+Data principles, this structure contains all the data needed for \doxylink{struct_e_c_u_component}{ECUComponent} functionality. ECU is a consumer-\/only component that receives and processes messages from other components (Camera, Lidar, INS).

Replaces inheritance-\/based data storage with pure composition.
