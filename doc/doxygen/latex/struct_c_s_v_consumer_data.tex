\doxysubsection{Detailed Description}
Data structure for CSV consumer component. 

Following EPOS Smart\+Data principles, this structure contains all the data needed for CSV consumer functionality. Tracks received CSV vehicle data messages and provides statistics about message reception.

CSV consumer processes messages with prepended timestamps and CSV records in the format\+: timestamp,id,lat,lon,alt,x,y,z,speed,heading,yawrate,acceleration
