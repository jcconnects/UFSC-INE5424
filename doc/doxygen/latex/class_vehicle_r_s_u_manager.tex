class Vehicle\+RSUManager$<$ Protocol\+\_\+\+T $>$}
\doxylink{class_vehicle_r_s_u_manager}{Vehicle\+RSUManager} -\/ Simplified collision domain logic

Leader selection is based on communication ability (collision domain). If a vehicle can receive STATUS messages from an \doxylink{class_r_s_u}{RSU}, they are within the collision domain and the \doxylink{class_r_s_u}{RSU} is eligible to be the leader.

The closest \doxylink{class_r_s_u}{RSU} (by distance) that we can communicate with becomes the leader.
