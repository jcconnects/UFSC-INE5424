Helper function to access TestComponentData from Agent.

Provides a way to access the test-specific data for verification in tests. This is a workaround since we can't directly access private component data.

Note: This is a test-only helper and should not be used in production code.

Returns
Pointer to TestComponentData (nullptr if not a test agent)
