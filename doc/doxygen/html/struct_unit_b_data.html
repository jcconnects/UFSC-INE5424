Detailed Description

Data structure for UNIT_B components.

Following EPOS SmartData principles, this structure contains all the data needed for both BasicProducerB and BasicConsumerB functionality. Replaces inheritance-based data storage with pure composition.

Uses a different default range (200.0f to 300.0f) compared to UNIT_A to provide clear differentiation between component types.
