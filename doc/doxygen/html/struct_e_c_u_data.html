Detailed Description

Data structure for ECU component.

Following EPOS SmartData principles, this structure contains all the data needed for ECUComponent functionality. ECU is a consumer-only component that receives and processes messages from other components (Camera, Lidar, INS).

Replaces inheritance-based data storage with pure composition.
